# Hackathon Tracks

The event offers multiple tracks. We are competing in the **NormaCore Track**.

## ⭐ Our track — NormaCore: "AI-Powered Robot Control"

> **Official description:** *Integrate the Station API into Codex or Claude so that it can control the
> robot by looking through the cameras.*

**Interpretation (important):** This does **not** mean "call an LLM API with a system prompt." It
means **make the Claude Code / Codex CLI itself the robot's brain** — using the agentic loop, with
the Station API exposed as tools (MCP). Finding the *application* is left to us — that is the whole
competition. See [`03-project-vision.md`](./03-project-vision.md) for the application we chose.

**Why we picked it:** It matches our strengths (agentic orchestration, full-stack, voice agents), it
uses the most advanced agentic coding tools, and the open-ended "find the application" framing lets
us build a real, fundable product rather than a tech demo.

## Other tracks (for reference / context)

### AI Track — "The All-in-One Dashboard for Robotic Founders"
*Sponsored by n8n, Lovable & Apify.* Build a dashboard that automates the business side of a robotics
startup end-to-end with n8n, Lovable, and Apify.

### Hugging Face Track — "Desk Hero"
Solve a simple everyday desk problem using **LeRobot** and any robot you want.

### ITQ Track — "Autonomous Detection and Collection of Bottle Caps with Computer Vision"
Complete a parkour within 5 minutes, avoid obstacle collisions, collect as many bottle caps as
possible. **Scored on:** caps collected, completion time, safe navigation. (This is the only track
with a published rubric.)

### MyBotShop Track — "Quadruped Track"
Conceptualize and pitch a startup idea built around quadruped robots (inspection, search & rescue,
agriculture, logistics, etc.).
